📊 Customer Segmentation Analysis Using RFM – Superstore Dashboard
--------------------------------------------------------------------------------
📌 Gambaran Proyek

Proyek ini bertujuan untuk menganalisis segmentasi pelanggan menggunakan metode RFM (Recency, Frequency, Monetary) berdasarkan dataset Superstore. Analisis divisualisasikan dalam bentuk dashboard interaktif menggunakan Power BI untuk memahami perilaku pelanggan dan mengidentifikasi segmen pelanggan yang paling bernilai bagi bisnis.

Metode RFM mengelompokkan pelanggan berdasarkan tiga indikator utama:

🕒 Recency → Seberapa baru pelanggan melakukan pembelian

🔁 Frequency → Seberapa sering pelanggan melakukan transaksi

💰 Monetary → Seberapa besar nilai pembelian pelanggan

Berdasarkan skor RFM, pelanggan diklasifikasikan ke beberapa segmen seperti Champions, Loyal Customers, Potential Loyalists, Promising, dan Needs Attention.

🛠 Tools & Teknologi
--------------------------------------------------------------------------------
Proyek ini menggunakan beberapa tools berikut:

📊 Power BI – untuk membuat dashboard visualisasi data

🧮 DAX (Data Analysis Expressions) – untuk menghitung skor RFM dan berbagai measure analisis

🗂 Superstore Dataset – dataset retail yang digunakan untuk analisis penjualan
